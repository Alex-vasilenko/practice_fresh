Индивидуальный подбор плана питания

План питания будет включать в себя:

  • Годовой сбалансированный рацион на 5 приёмов пищи в день. Базовая версия рациона адаптирована для людей по бюджету и составу (без киноа и лосося). Расширенные варианты рациона вариативны по бюджету и ограничениям;
  • Расчёт КБЖУ. Норму и план потребления с рекомендациями от платформы;
  • Расчёт приёма витаминов, микроэлементов к годовому рациону;
  • Норму водного баланса и питьевой режим.

Онлайн тестирование

За 20 минут узнаем о вашем текущем образе жизни. Вы ответите на вопросы, и мы определим ваши физиологические особенности, текущий пищевой и питьевой режим, а также физическую активность.

Модуль анализа

Проанализируем текущий рацион, поможем сформулировать ваши цели и выявить нормы питания.

Ответы на вопросы

What is Web Development?
Web Development broadly refers to the tasks associated with developing functional websites and applications for the Internet. The web development process includes web design, web content development, client-side/server-side scripting and network security configuration, among other tasks.
What is HTML?
HTML, aka HyperText Markup Language, is the dominant markup language for creating websites and anything that can be viewed in a web browser.
What are some basic technical skills of a Front-End developer?
  • HTML, CSS, JavaScript
  • Frameworks (CSS and JavaScript frameworks)
  • Responsive Design
  • Version Control/Git
  • Testing/Debugging
  • Browser Developer Tools
  • Web Performance
  • SEO (Search Engine Optimization)
  • Command Line
  • CMS (Content Management System)
What is HTTP?
HTTP, aka HyperText Transfer Protocol, is the underlying protocol used by the World Wide Web and this protocol defines how messages are formatted and transmitted, and what actions Web servers and browsers should take in response to various commands.
What is CORS?
CORS, aka Cross-Origin Resource Sharing, is a mechanism that enables many resources (e.g. images, stylesheets, scripts, fonts) on a web page to be requested from another domain outside the domain from which the resource originated.
